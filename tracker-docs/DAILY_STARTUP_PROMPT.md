# Daily Startup Prompt

Copy and paste this into a fresh Claude Code session to resume work on the tracker:

```text
Please read:
- tracker-docs/README.md
- tracker-docs/CURRENT_TASK.md
- tracker-docs/DECISIONS.md
- tracker-docs/TODO.md
- tracker-docs/SESSION_LOG.md

Then:
1. Summarize the current state of the app in a few sentences.
2. Tell me the next best step.
3. Continue the work.
4. Update CURRENT_TASK.md and SESSION_LOG.md as needed while you work.
```

## Optional Add-On

If you already know the focus, add one line like:

```text
Today's focus is adding voice input to the log entry modal.
```
